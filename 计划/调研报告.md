### A3. 研究方向全景

四个方向（按重要性排序）：

#### 方向 1（最重要）：阿秒时间分辨电子动力学
- **核心问题**：阿秒尺度（10⁻¹⁸ 秒）下电子怎么运动？
- **方法**：高次谐波产生（HHG）+ 阿秒脉冲 + 泵浦-探测技术
- **代表**：Time-Resolved Multielectron Coincidence Spectroscopy（多电子符合谱）
- **核心论文**：
  - PRL 132, 083201 (2024) — 时间分辨多电子符合谱（气相 Xe）
  - PRL 130, 153201 (2023) — 自电离共振对波包动力学的影响（N₂）

#### 方向 2：液体水和水溶液的超快动力学
- **核心问题**：液体中能量和电荷怎么传递？液相化学和气相有什么本质区别？
- **关键现象**：
  - **ICD (Intermolecular Coulombic Decay)**: 分子间库仑退激
  - **液体水光电子能谱**：电子在液相中的散射行为
- **代表论文**：
  - Nat. Commun. 16, 6732 (2025) — ICD 与质子转移竞争
  - Nat. Chem. 14, 1126-1132 (2022) — 气相 vs 液相光致异构化时间尺度
  - Nat. Commun. 16, 7211 (2025) — 分子电荷转移中 1.5-fs 延迟
  - Nat. Photon. 19, 847 (2025) — Fano 共振波包相位
  - PRL 128, 133001 (2022) — 液体水中 ICD（编辑推荐）

#### 方向 3：新型阿秒光源研制
- **目标**：更便携、稳定、白光覆盖的阿秒光源
- **关键技术**：固体 HHG、桌面式 EUV、液相 HHG 装置
- **代表论文**：
  - Phys. Rev. Res. 8, 013067 (2026) — 固体白光阿秒脉冲
  - Rev. Sci. Instrum. 97, 013005 (2026) — 液相 HHG 装置
  - Ultrafast Sci. 6, 0147 (2026) — SF6 振动量子拍频（封面）
  - 物理学报 75(10): 100405 (2026) — 桌面式极紫外 HHG 综述

#### 方向 4：关联电子材料阿秒动力学（最新方向）
- 与孟胜（理论）合作。属于还在起步的方向。

### B2. 详细分析：SciAtlas

**为什么最相关**：
- 内置 `sciatlas researcher-review` 子命令——直接产出研究者背景报告
- 内置 `sciatlas literature-review` 子命令——产出文献综述
- 内置 `sciatlas trend-report` 子命令——产出趋势分析
- 内置 `sciatlas related-authors` 子命令——找该老师相关同行

**产出格式**：
- 机器可读：`request.json` / `response.json`（可二次加工）
- 人类阅读：`summary.txt` / `report.md`

**对我们的启示**：
- 我们可以**基于 SciAtlas 的 researcher-review 命令**起步，但它的输出是英文且未结构化到学习讲义层面
- 需要在我们的工具里加一道"翻译+结构化"流水线，把它转成学习讲义模式

### B3. 详细分析：paper-to-course

**为什么高度相关**：
- "直觉先于形式化"——和我们 `读者画像与叙事基调.md` 的"起承转合"原则一致
- Module 0 (背景与前置) 必选——直接对应我们"已有基础"调研
- 双语术语格式 `中文解释（English Term）` ——可直接采纳
- 8 模块结构（背景 → 问题 → 脉络 → 核心洞见 → 方法 → 实验 → 结果 → 启示）——和我们计划的"五部结构"基本一致

**对我们的启示**：
- 学习讲义的**单论文讲义版**可以参考这个
- 但我们要做的是"从基础到这位老师前沿论文的完整路径"，比 single paper-to-course 更长程
- 必须把 path 扩展为多论文或多章节层次

### B4. 综合结论

**我们工具的差异化定位**：

| 维度 | SciAtlas | paper-to-course | **我们的工具** |
|:-----|:---------|:----------------|:--------------|
| 输入 | 论文 / 作者名 | 论文 PDF | **老师主页 URL** |
| 输出 | 文献综述 | 单论文讲义 | **信息卡 + 完整入门讲义** |
| 学习路径 | 单论文级 | 单论文级 | **从大二基础到老师最新前沿的完整路径** |
| 跨领域复用 | 是 | 是 | **是**（输入新老师即重生成）|
| 教学法框架 | 无 | 直觉优先 | **起承转合五步叙事流**（自有方法论）|

**结论**：
1. **不要从零造**——信息卡部分可以用 SciAtlas 起手
2. **学习讲义部分可以借 paper-to-course 的设计思路**，但要扩展到完整路径
3. 依赖图自动化可借鉴 InstructKG
4. 关键差异化在：从零起步 + 跨老师复用 + 教学法框架
