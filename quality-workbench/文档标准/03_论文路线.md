# 03 论文路线

状态：只保留讨论入口，尚未展开。

## 文档职责

`03_论文路线.md` 是论文定位工具。它回答“导师在当前方向具体做什么，相关论文之间如何构成研究路线”。

它要帮助学生判断一篇论文在当前路线中承担什么角色。正文要避免两种写法：一是论文摘要列表，二是只按年份排列的论文清单。

## 整体大纲

待讨论。先确定整篇怎样从“当前问题”走到“论文角色”，再决定哪些表格或图能保留。

## 小节设计

讨论每个小节时先看四件事：

- 这一节让读者做什么。
- 为什么要这样设计。
- 读完后应留下什么判断。
- 哪些地方最容易写偏。

## 图文设计

待讨论。重点先判断论文之间到底是问题链、平台链、方法比较，还是几条并行线索。

已暴露的风险：

- 横向 Mermaid 流程图容易过长，显示变小。
- 论文路线不一定适合画成流程图。
- 更可能需要“论文 -> 问题 -> 方法 -> 角色 -> 学习缺口”的矩阵。

## 待确认问题

- 如何讲清论文之间的关系，而不是给论文贴标签。
- 何时用问题链，何时用矩阵表。
- 目标论文和回看论文如何服务 `04`，但不抢 `04` 的学习设计。
- 旁支和弱证据如何呈现。
