<a id="taste-main-question"></a>

## 中心问题

文稿品味先体现在读者能不能跟着走。很多材料看起来完整，读者却只记住一堆字段、术语和图。问题通常不在句子不够漂亮，而在资料没有被加工成路径。

本文件只解决一个问题：

```text
怎样让每一节、每一段、每张图都把读者往同一个出口推？
```

导师材料面向新手。新手需要先知道自己在看什么，再知道为什么要看这一块，随后把事实、领域、论文和学习任务接起来。文稿设计的任务，就是让这种接力稳定发生。

<a id="taste-path"></a>

## 从资料堆到读者路径

写正文前，先写四句话。它们比目录更早：

```text
读者进来时已经知道什么：
他此刻最容易卡在哪里：
本文用哪几步带他跨过去：
读完后他能说出或做出什么：
```

这四句话用于筛掉多余内容。比如 `02_领域地图.md` 不能只写“介绍导师研究领域”。更合适的路径是：

```text
读者已经从 01 拿到履历和论文集合，但不知道论文里的方向词属于哪个大领域。
他容易把材料名、方法名、问题名混成一团。
本文先给范围感，再收束到导师当前问题域，最后把少数关键概念交给 03。
读完后，他能说出导师方向大致处在哪片领域，旁边哪些方向容易混淆。
```

路径写清后，再排目录。目录只是路径的外壳；路径没定，目录越细越像资料库。

<a id="taste-handoff"></a>

## 章节怎样接力

章节接力要能回答四个问题：

| 位置 | 要回答的问题 | 失败信号 |
|:---|:---|:---|
| 入口 | 上一节交给我什么 | 本节删掉上一节也能独立摆着 |
| 推进 | 我让读者多理解什么 | 本节只是在补资料 |
| 局部支撑 | 哪个关系需要图、表、例子或证据说明 | 读者要在脑子里硬扛关系 |
| 出口 | 下一节会拿走什么 | 下一节不能使用本节结果 |

这张表也约束支撑方法本身。支撑方法如果只列原则，不给执行者一个可使用的出口，也会变成资料堆。

<a id="taste-paragraph"></a>

## 段落怎样承担任务

段落只承担一个主要动作。常见动作包括：

```text
承接、定位、解释、比较、举例、标证据边界、教读图、交给下一节
```

一个段落里同时塞定义、历史、例子、结论和建议，读者会失去重心。写完后逐段问：

```text
这段接住了前面什么？
这段让读者多理解什么？
这段里的判断证据够不够？
这段有没有把并列关系压成一长串？
下一段会怎样使用它？
```

如果一段只能回答“它让文章显得完整”，就删掉或移入内部材料。

<a id="taste-support"></a>

## 图表和证据怎样嵌入

图、表和证据说明只在局部卡点出现。它们的顺序应当是：

```text
正文提出学生问题
  -> 图表或证据说明回答这个问题
  -> 图旁写清读法或证据边界
  -> 后文消费这个结果
```

表格适合重复字段和并排比较，不负责讲主线。图适合让学生看见空间、结构、路线或证据边界，不负责替正文下判断。证据说明要贴近关键判断，不能只堆在文末。

常见坏味道：

| 坏味道 | 读者感受 | 修法 |
|:---|:---|:---|
| 报菜名 | 知道列了很多东西，不知道为什么这样排 | 先写章节接力，再删掉接不上主线的项 |
| 章节孤岛 | 每节都能懂一点，合起来没印象 | 给每节补入口和出口 |
| 表格过密 | 扫字段很快，但脑子里没有链条 | 表格只放字段，主线解释回到正文 |
| 总览图孤立 | 开头图能看，后文又全靠想象 | 在具体卡点处放局部支撑 |
| 证据漂浮 | 文末有来源，正文判断不知道靠哪条 | 判断附近标证据强弱 |
| 模板腔 | 换个导师也能套进去 | 加入本导师、本学科、本证据链里的具体动作 |

## 连续改造示例

坏稿常见写法：

```text
导师方向涉及凝聚态物理、超快光谱、强关联体系、非平衡动力学等内容。
这些方向在现代物理中十分重要，并与材料科学、量子调控和新型功能材料有关。
```

这段的核心问题是缺少关系。学生只得到一串名词。

先写接力：

```text
上一节给我：履历和论文集合中反复出现的方向词。
本节推进：把方向词放进大领域，让学生知道哪个是范围、哪个是方法、哪个是问题域。
局部支撑：如果文字压不住范围感，用范围定位图；如果只是上下位关系，用短收束链。
下一节拿走：03 可以沿着少数关键概念进入论文问题链。
```

再写正文：

```text
从 01 的履历和论文集合看，学生会先碰到一串相近的词：凝聚态物理、超快光谱、强关联体系、非平衡动力学。这里先不急着解释每个词的细节，先把它们放回专业地图。

凝聚态物理是较大的学科范围，关注大量粒子组成的材料和物质状态。超快光谱更像观察工具，用很短的光脉冲追踪材料在极短时间内怎样响应。强关联体系和非平衡动力学更靠近问题域：材料内部粒子相互作用很强，受到激发后会出现难以用简单模型解释的动态行为。

因此，本节先给一个粗略位置感：导师方向可先放在凝聚态物理里的超快物质科学附近阅读，后续论文路线会沿着“用超快方法观察复杂材料响应”这条线继续往下走。
```

改造后的段落没有追求华丽。它让学生拿到三个可用结果：大领域、方法入口、问题域。

<a id="taste-quality-check"></a>

## 质量检查

每份文稿进入工作台基线前，至少过这组检查：

| 检查 | 通过标准 |
|:---|:---|
| 读者路径 | 能写出读者起点、卡点、推进步骤和出口 |
| 章节接力 | 每节都能说明上一节给什么、下一节拿什么 |
| 段落任务 | 每段只做一个主要动作 |
| 局部支撑 | 图、表、例子、证据说明出现在读者真正卡住的位置 |
| 证据语气 | 直接事实、交叉判断、弱线索和待复核没有被同一种语气抹平 |
| 新手入口 | 术语和缩写第一次出现时有足够上下文 |
| 读者输出 | 文末或关键段落后有可复述、可回看的小出口 |
| 文风 | 没有空泛开场、机械转折、堆砌式总结 |

这组检查只能兜底。真正的文稿质量要回到学生输出：学生能否用自己的话复述本文出口，并知道卡住时回看哪里。
