# 03 论文路线

状态：待讨论。

## 文档职责

`03_论文路线.md` 是论文定位工具。它回答“导师在当前方向具体做什么，相关论文之间如何构成研究路线”。

它不应写成论文摘要列表，也不应只按年份排列。它要帮助学生判断一篇论文在当前路线中承担什么角色。

当前已暴露的问题：

- 横向 Mermaid 流程图容易过长，显示变小。
- 论文路线不一定适合画成流程图。
- 更可能需要“论文 -> 问题 -> 方法 -> 角色 -> 学习缺口”的矩阵。

## 待确认问题

- 如何讲清论文之间的关系，而不是给论文贴标签。

- 目标论文和回看论文如何服务 `04`，但不抢 `04` 的学习设计。
- 旁支和弱证据如何呈现。
