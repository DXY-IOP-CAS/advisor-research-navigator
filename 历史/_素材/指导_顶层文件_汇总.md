## 一、学习方法论体系

### 总体设计

**一句话本质**：一次只学一个东西。讲一小段 -> 用户实操一小段 -> 用户复述 -> 确认理解 -> 下一个。（来源：学习方法论.md 2.1 节）

**设计依据**（4 条文献支撑）：
- 认知负荷理论（Sweller 1988）：工作记忆容量有限（约 4-7 块），一次塞多个导致过载
- 工作样例效应（Sweller & Cooper 1985）：先看完整样例再动手，耗时减半、错误率降至 1/5
- 自我解释效应（Chi et al. 1989）：复述环节强制自我解释
- 检索练习效应（Roediger & Karpicke 2006）：闭卷回忆远优于重复阅读

### 双轨并行

| 维度 | 操作型学习 | 调研型学习 |
|------|-----------|-----------|
| 目标 | 会用这个工具/技能 | 理解设计决策，形成判断 |
| 初体验 | 跑一次，看输入输出 | 读摘要/README |
| 知识点形态 | 概念 + 沙盒实操 + 验证 | 概念 + 对比分析 + 判断问题 |
| 复述重点 | "跑出什么了？为什么？" | "设计决策对我们意味着什么？" |
| 模块收尾 | 综合应用（闭卷实操） | 收尾盘点（回顾所有判断） |

判定标准：模块中有无可运行的东西。有 -> 操作型路线；没有 -> 调研型路线。（来源：学习方法论.md "双轨学习"节）

### 五阶段流程

操作型：`全景图 -> 初体验 -> 定大纲 -> 逐点学习 x N -> 综合应用`

调研型：`全景图 -> 快速定位 -> 定大纲 -> 逐点学习 x N -> 收尾盘点`

**阶段-模式绑定**（每个阶段绑定一种 AI Agent 设计模式）：
- **全景图 + 初体验**：Plan-and-Execute（先全貌后细节，不跳步）
- **定大纲**：Plan-and-Execute（拆分学习点等待审核，HITL 关键节点）
- **逐点学习（调研型）**：ReAct（搜索->阅读->分析->再搜索）
- **逐点学习（操作型）**：Plan-and-Execute（按沙盒步骤执行）
- **收尾盘点（调研型）**：Reflection + HITL（自评回顾，人工确认闭合）

模式来源：Anthropic (2024.12) Building Effective Agents + 行业共识 (2026)。（来源：学习方法论.md "阶段-模式绑定"节）

### AI 执行 8 条规则

1. **用户控速**：不得自行推进，用户说"继续"是唯一信号
2. **零跳步**：不省略中间推理，禁用"显然""易得"
3. **文件出入**：首次教学（全景图/定大纲/逐点课件）+ 重大修正 -> .md 文件，对话告知文件名+概要。输入写完整可复制字符串，输出不省略中间步骤
4. **按需验证**：不复述中展示正确理解 -> 直接推进；偏差 -> AI 给 1 个针对性追问
5. **防幻觉**：arXiv 编号/论文标题/作者名/DOI/公式/物理常数/实验数据/第三方引用/工具行为 -> 必须先 WebSearch + WebFetch 验证再写入
6. **原始来源回溯**：调研型课件涉及外部框架/论文/工具 -> 必须先读原始来源再写课件
7. **反谄媚分析**：调研型课件对照外部框架时，必须同时含"支持"和"挑战"两列
8. **追问回路**：用户主动追问 -> AI 全力解答 -> 解答完后主动确认是否回到主线

### 协作规则

AI 写文件 -> 告知文件名+概要 -> 用户读 -> AI 给引导性提问 -> 用户复述+指不懂 -> AI 纠正/补充 -> 确认 -> 继续。（来源：学习方法论.md "协作规则"节）

### 沙盒模式

**三条原则**：
1. **极简沙盒**：最多两个文件（`_沙盒_` + `_结果_`），放在模块文件夹内与课件同级
2. **人触发，人解释**：AI 只生成材料，用户亲手执行。用户先解释，AI 禁止在用户复述前给结构化分析
3. **不重复聊天**：复述在对话中完成

**沙盒分级**：
- 快速演示：小概念，对话中展示 30s-2min，不落盘
- 物理沙盒：核心机制需要亲手体感，两个文件 5-15min，落盘

（来源：学习方法论.md "沙盒模式"节）

### 联网搜索铁律

凡涉及论文引用、公式、定理、实验数据、历史事实、统计数字、API 行为、时效性信息 -> 必须先 WebSearch 验证再输出。间接引用必须追溯原始论文。搜索过的 URL 附文件末尾"验证来源"小节。

**搜索回退链**（硬约束，严格依序，禁止跳步）：
```
原生 WebSearch -> 连续 2 次失败 -> Serper Google Search -> 连续 2 次失败 -> Tavily Search
原生 WebFetch -> 连续 2 次失败 -> Serper Scrape -> 连续 2 次失败 -> Tavily Extract
```
- "失败"定义：工具返回错误，或搜索结果完全为空（0 条有效结果）
- 单次报错不计为失败，必须重试 1 次
- 连续 2 次相同错误才可认定该级不可用
- 禁止以"图方便""省时间"跳过任何一级
- 违反本链的输出，用户有权要求删除全文重新搜索

（来源：身份与学术标准.md "联网搜索铁律"节）

**流水线 C：撰写一章讲义**
```math
① arxiv 搜索主题相关论文
② 翻译关键论文
```

## 五、李自翔学术全景

### 学术编年三阶段

**阶段一（2015-2019）：绝对防守——对称性消去符号问题**
- 博士/博士后初期，在清华高等研究院
- 核心贡献：提出马约拉纳表象（Majorana Representation），将费米子算符拆为两个实数算符，使一类排斥相互作用模型变得无符号
- 建立 MTR 对称性理论（Majorana-Time-Reversal Symmetries）：证明若哈密顿量满足两个相互反对易的 MTR 对称性，费米子双线性矩阵行列式恒正
- 观测到"涌现超对称"（Emergent Supersymmetry）——凝聚态晶格模型中首次数值观测
- 代表作：PRB 92, 081105(R) (2015, 被引 200+)、PRL 116, 250601 (2016, 被引 120+)、Science Advances 4, eaau1463 (2018)

**阶段二（2019-2022）：热力学折中——伯克利博后期间**
- 加州大学伯克利分校博后
- 发现"渐近无符号"（Asymptotic Sign Free, ASF）：狄拉克费米子系统中，增大系统尺寸后符号问题反而"自愈"
- 研究方向拓展到非常规超导、电声子耦合
- 代表作：npj Quantum Materials 6, 38 (2021)、PRL 127, 247203 (2021)、arXiv:2211.00663 (2022)

**阶段三（2023-2026）：降维截胡——独立建组后的最新突破**
- 独立建组，算法深度迭代
- 虚时非平衡动力学 QMC：放弃无限投影时间的执念，在短时演化阶段（符号灾难暴涨前）提取量子临界指数
- 虚时 Mpemba 效应：选择"更远"的初始态反而更快趋近基态，缩短投影时间
- 拓展到准晶超导、SMG（对称性质量产生）、非厄米系统
- 代表作：Nature Physics 19, 1423 (2023)、PRL 132, 176501 (2024)、Science Advances 12, eadz4856 (2026)、PRL 136, 086502 (2026)、PRL 136, 100403 (2026)

### 对导师调研工具的可参考价值

李自翔学术全景为用户决定了**调研目标导师**的完整学术画像。对调研工具的具体参考：

3. 论文分类维度：按时间线 + 按物理问题（符号问题突破路线）+ 按研究方法（对称性/RG/动力学）

6. 21 篇论文覆盖 2014-2026 共 12 年，产出频率约 1.75 篇/年——属于高产，调研工具应能识别此类产出节奏特征与趋势
