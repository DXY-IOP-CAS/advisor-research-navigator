# 旧入口：材料怎么读

冻结说明：

- 来源：早期学习 `00_材料导读` 时使用的入口文件。
- 保存原因：保留当时“先学会怎么读整套材料”的历史路线。
- 当前替代文件：`../../00_入口/00_学习路线.md` 和 `../../00_入口/01_方向入门讲义.md`。
- 是否可作为主阅读：不可。当前主阅读从 `00_入口/` 和 `02_分讲/` 开始。
- 不应如何使用：不要再把它当成第一课，也不要把学习重心带回“五份文档怎么设计”。

这是一份历史说明，不是当前学习入口。

早期学习时，这个文件曾用来解释怎样读 `00_材料导读.md` 和整套 `00-04` 材料。后来学习重心已经转向张鹏举老师的研究方向，当前入口改为：

- `../../00_入口/00_学习路线.md`
- `../../00_入口/01_方向入门讲义.md`

所以这份旧入口退出主阅读路径，只保留它当时解决过什么问题。

## 当时想解决的问题

早期最大的问题不是“张老师研究什么”，而是“这五份材料该怎么用”。当时需要先知道：

- `00_材料导读` 是读法说明，不是研究方向正文；
- `01_基础画像` 先建立事实底座；
- `02_领域地图` 负责解释领域位置；
- `03_论文路线` 负责从论文看问题链；
- `04_学习向导` 负责把学生已有基础接到后续学习。

这个入口当时有用，因为学习者还不熟悉五份材料的分工。

## 为什么现在不再作为入口

现在用户已经熟悉 `00_材料导读` 的作用，继续把“怎么读五份文档”放在入口，会把学习拉回文档设计问题。

当前真正要学的是：

```text
张鹏举老师研究什么；
为什么要用阿秒、XUV、HHG、PES 这些工具；
液体水 ICD 为什么适合作为入口；
哪些论文能代表当前主线；
学生需要补哪些基础才能读论文。
```

所以旧入口不再进入 `00_入口/`。需要追溯早期学习路线时，看这份历史说明即可。
