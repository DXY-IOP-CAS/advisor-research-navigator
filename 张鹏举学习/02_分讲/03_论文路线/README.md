## 原文主线

`03_论文路线.md` 的主线不是按顺序介绍所有论文，而是把论文变成问题链：哪些论文提供液相信号底座，哪些论文推进液体水 ICD 机制，哪些论文扩展到分子阿秒动力学，哪些论文属于平台建设或正在形成的分支。

主线可以压成：

```text
论文表只是材料池
  -> 先找当前主线问题
  -> 区分底座论文、机制论文、平台论文、回看论文和分支论文
  -> 用液体水 ICD 建立第一个起步闭环
  -> 再扩展到分子阿秒、Fano 波包、电荷转移和平台边界
```

## 主线学习计划

1. 论文表怎样从材料池变成问题链  
   学完要能说：论文路线不是论文摘要列表，而是看每篇论文在问题推进中承担什么角色。

2. 为什么液体水 ICD 是起步闭环  
   学完要能说：它同时连接液相光电子谱、电子-电子符合测量、低能电子、同位素差异和机制判断。

3. 2022 与 2025 液体水 ICD 论文怎样前后推进  
   学完要能说：2022 解决直接识别，2025 解决机制竞争。

4. 分子阿秒和 Fano/电荷转移论文怎样扩展问题域  
   学完要能说：这些论文把电子时间、相位和电荷转移问题接到更一般的分子动力学。

5. 平台论文怎样打开后续问题  
   学完要能说：光源、样品和谱仪不是附属设备，而是决定哪些机制问题能被看见。
