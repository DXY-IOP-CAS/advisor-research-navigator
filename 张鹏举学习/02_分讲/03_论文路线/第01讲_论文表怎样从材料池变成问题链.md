# 第 01 讲：论文表怎样从材料池变成问题链

这一讲先解决一个读论文路线时最容易乱的问题：不要把论文表看成“论文越多越好”的清单。对新手来说，论文表只有在能回答“这篇论文承担什么角色”时，才真正有用。

对应原文：`../../01_原文/03_论文路线.md` 中 `资料概览`、`从领域问题进入论文`、`论文篮`、`主线问题`。

本讲只解决：

```text
怎样把论文从材料池，读成一条问题推进路线。
```

本讲暂时不解决：

```text
不细读单篇论文图，不展开 Fano / CF3I / 固体阿秒，不安排课程学习，不评价论文好坏。
```

读完要能说：

```text
论文路线不是论文摘要列表，而是看每篇论文让哪个问题变得更可测、更可分辨或更可解释。
```

## 论文表不是代表作排行榜

看到一堆论文时，最容易犯的错是按题名、年份、期刊或引用数排序。这样很快会乱，因为不同论文解决的问题不在同一层。

在这份 `03_论文路线.md` 里，论文表先被当作“材料池”。材料池的意思是：这些论文都可能有用，但它们不承担同一种任务。

有些论文是目标入口，有些是前置证据，有些是底座，有些是平台，有些只是相邻语言。读的时候先问角色，不先问排名。

## 先用三个问题筛论文

原文给出的入口问题可以压成三句：

```text
这个电子信号可信吗？
这个信号能分辨什么机制？
实验平台能不能支撑后续问题？
```

这三句分别对应三层：

| 层次 | 问什么 | 典型论文 |
|:---|:---|:---|
| 可测 | 液体里出来的电子信号能不能定标、解释、修正 | 液体水电离能、可调 XUV 液体水 PES、低能电子分布 |
| 可分辨 | 能不能把某个机制从背景里识别出来 | 2022 液体水 ICD |
| 可解释 | 信号背后的机制怎样竞争或演化 | 2025 液体水 ICD 竞争机制 |

这就是论文路线的第一条主干。

## 目标论文、回看论文、底座论文

先看液体水 ICD 这条线。

目标论文是 2025 年液体水 ICD 竞争机制论文。它适合作为入口，不是因为它最新，而是因为它把信号、机制和对照压在一起：

```text
H2O/D2O 符合测量
  -> ICD 相对效率差异
  -> 质子转移和非绝热弛豫竞争
```

回看论文是 2022 年液体水 ICD 首次观测论文。它解决的是前置问题：

```text
液体水里能不能直接识别 ICD？
```

底座论文包括液体水电离能、可调 XUV 液体水光电子谱、低能电子分布。它们不一定最像“机制突破”，但它们回答了更基本的问题：

```text
液体里的电子信号到底能不能相信？
低能电子出来以后会不会被散射和逃逸过程改坏？
```

所以这三类论文不是平行堆放，而是有依赖关系：

```text
底座论文：让液体电子信号可信
  -> 回看论文：让 ICD 从背景里可识别
  -> 目标论文：解释 ICD 和其他过程怎样竞争
```

## 分子阿秒和平台论文先放在旁边

原文还列了分子阿秒、Fano 波包、CF3I 电荷转移、平台论文。它们重要，但不是当前第一入口。

分子阿秒论文提供的是相邻语言：

```text
时间
相位
波包
电荷转移
多电子相关
```

平台论文提供的是实验边界：

```text
光源够不够
样品能不能进真空
电子能不能收集
低能背景能不能分开
```

现在先知道它们的位置，不急着读细节。否则会从液体水 ICD 一下跳到 Fano、固体阿秒和液相 HHG，主线又会散。

## 这一讲的路线图

把 `03_论文路线.md` 的第一层读法压成一条链：

```text
不要把论文表当排行榜
  -> 先问论文承担什么角色
  -> 液体水 ICD 线先按“可测、可分辨、可解释”读
  -> 分子阿秒论文先作为相邻语言
  -> 平台论文先作为实验边界
```

这条链的出口是：

```text
我知道为什么先读液体水 ICD 线，而不是把所有论文一起读。
```
