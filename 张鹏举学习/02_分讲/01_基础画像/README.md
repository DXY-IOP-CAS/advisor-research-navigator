这组分讲只解决事实底座问题：我们讨论的是谁，资料从哪来，履历阶段怎么读，论文集合有什么边界。

## 原文主线

`01_基础画像.md` 的主线不是评价导师，也不是直接判断当前研究方向。它先让学习者知道：讨论对象是谁，公开资料从哪里来，履历阶段怎样变化，论文集合有什么边界，哪些信息还需要后续核查。

主线可以压成：

```text
确认讨论对象
  -> 看资料来源能证明什么
  -> 按履历阶段理解方向变化
  -> 把论文表当成材料池和边界，不当成代表作清单
  -> 把当前方向判断交给 02_领域地图 和 03_论文路线
```

## 主线学习计划

1. 先看事实底座  
   学完要能说：`01_基础画像` 先确认身份、来源和材料边界，不直接评价导师。

2. 履历阶段怎么读  
   学完要能说：早期论文不能直接等于当前主线，但能作为前史理解方向怎样长出来。

3. 论文集合边界怎么读  
   学完要能说：论文表是候选材料池，不是精选代表作清单；后续要用原文、图表和论文关系继续筛选。

## 允许发散但要回到主线
