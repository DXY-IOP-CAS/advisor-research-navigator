## 原文主线

`04_学习向导.md` 的主线不是列课程表，而是从目标论文倒推最小学习路径。先选一个能承接当前方向的目标论文，再判断读这篇论文需要哪些基础、哪些图先看、哪些问题可以进组前准备。

主线可以压成：

```text
先锁定目标论文和目标图
  -> 判断读论文需要哪些基础概念
  -> 从本科基础补到光电子谱、液相样品、符合测量和 ICD
  -> 用一篇论文做起步闭环
  -> 形成能请教导师或师兄师姐的问题
```

## 主线学习计划

1. 从本科基础到第一篇目标论文，中间缺什么  
   学完要能说：补课不是泛泛补全部物理，而是围绕目标论文反推最小知识链。

2. 目标论文的图先怎么看  
   学完要能说：先看横轴、纵轴、信号强度和每张图回答的问题，不急着推公式。

3. 液相 PES、符合测量和 ICD 需要哪些前置概念  
   学完要能说：为什么需要能量、谱峰、束缚态/连续态、散射、低能电子和同位素效应这些基础。

4. 怎样把学习输出变成进组前问题  
   学完要能说：能围绕目标论文提出具体问题，而不是只说“我想学阿秒”。

## 进入条件

进入这里前，至少要完成两件事：

1. 在 `03_论文路线` 里初步看出哪些论文最适合作为学习入口。
2. 能说清目标论文大概需要哪些概念，例如阿秒/XUV、光电子谱、ICD、液相样品、符合测量、电子弛豫和电荷转移。

如果目标论文还没选出来，学习向导只能写成泛泛课程清单，不适合作为当前主线。
