# 04_学习向导主线表

这张表从 `../../01_原文/04_学习向导.md` 抽出。它的任务是把 `03_论文路线.md` 交下来的目标论文、核心图、基础缺口和平台链路，变成进组前可执行的学习路线。

## 原文到底在做什么

`04_学习向导.md` 回答的是：

> 一个已经粗读事实、领域和论文路线的学生，怎样从高数、线代和普通物理出发，补到能读懂目标论文入口、核心图、平台链路，并能提出可请教的问题。

它不是课程表，也不替代教材。它按照“读目标论文时会做什么动作”来组织学习。

## 整份文档学习顺序

```text
先站稳缩写和学习目标
  -> 再用目标论文图反推读图动作
  -> 再把卡点映射到数学、物理、量子、谱学和平台基础
  -> 再形成液体水 ICD 起步闭环
  -> 再补 PES、TRPES、阿秒、液相复杂环境和平台边界
  -> 最后回到论文路线，形成短汇报和可请教问题
```
