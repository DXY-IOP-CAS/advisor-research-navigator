# 00_材料导读主线表

这张表从 `../../01_原文/00_材料导读.md` 抽出。它的任务不是补专业概念，而是把五份成品文档变成一条可学习、可回退、可核查的路线。

## 原文到底在做什么

`00_材料导读.md` 回答的是：

> 第一次接触张鹏举老师方向的学生，应该按什么顺序读这套材料，怎样理解引用和证据标记，卡住时回到哪份文档，最后怎样把论文、核心图和平台链路转成可请教的问题。

它不是导师评价，也不是论文综述。它的核心作用是防止读者一上来陷进单篇论文或单个术语，而忘了这套材料本来是一个五步认知阶梯。

## 整套五文档学习顺序

按导读重启时，学习顺序是：

```text
00_材料导读：先学会怎样读这套材料
  -> 01_基础画像：锁定身份、履历、论文集合和来源边界
  -> 02_领域地图：建立张老师当前方向的领域坐标
  -> 03_论文路线：把论文群读成问题链
  -> 04_学习向导：把目标论文和平台链路倒推成学习路径
  -> 回看 01：用更深理解复核事实底座
```

本次重启学习应先讲 `00` 的这条总路线，再进入 `01`。不要直接续讲旧的 `02_领域地图` 局部例子。
