# 张鹏举学习

这个文件夹当前用于学习张鹏举老师的研究方向和相关专业知识。更一般地说，它是一个“导师方向学习区”的实例：以后换成其他导师，也应沿用这里的学习方法。

它不是项目设计工作台，不承担正式成品开发、规则回收、模板修改或验证器改造任务。

现在的规则很简单：想学习就走入口和分讲；想核对事实就看原文；想追溯某次问答才看记录；想了解文件夹规范和历史迁移才看 `90_规范与历史/`。

## AI 接手先做什么

新来的 AI 不要直接凭分讲标题开始讲概念。必须先读 [90_规范与历史/00_学习区规范.md](90_规范与历史/00_学习区规范.md)，按里面的固定流程做：

```text
先锁定本轮原文档
  -> 从原文档抽出整文学习地图
  -> 按主线拆子任务
  -> 每个子任务里可以讨论和发散
  -> 发散后必须回到原文主线
```

分讲只是帮助理解原文主线的脚手架，不能替代 `01_原文/` 里的原成品文档。

这里的“整文学习地图”必须先回答：这份成品文档的主问题是什么，全文有哪些重要内容块，哪些只是入口例子，内容块之间怎样由浅入深地学，学完整份文档后学习者要能说什么、判断什么或做什么。没有这张地图，不要直接讲某个概念、论文或图。

## 先从哪里读

第一次回来继续学，按这个顺序走：

1. 读 [00_入口/00_学习路线.md](00_入口/00_学习路线.md)，知道当前学到哪里、下一步该读什么。
2. 读 [00_入口/01_方向入门讲义.md](00_入口/01_方向入门讲义.md)，先把张老师方向压成一张总览图。
3. 进入 [02_分讲/](02_分讲/) 逐讲学习。分讲才是之后主要新增和复习的地方。

如果你只是想接着上次继续，当前最合适的下一步是：

- [02_分讲/02_领域地图/第01讲_张老师到底研究什么.md](02_分讲/02_领域地图/第01讲_张老师到底研究什么.md)
- [02_分讲/02_领域地图/第02讲_为什么基础物理要走向极端条件.md](02_分讲/02_领域地图/第02讲_为什么基础物理要走向极端条件.md)
- [02_分讲/02_领域地图/第03讲_液体水 ICD 为什么适合作为入口.md](02_分讲/02_领域地图/第03讲_液体水%20ICD%20为什么适合作为入口.md)

如果目标是系统学会五份成品文档，先不要从某一讲继续。应让 AI 逐份生成整文学习地图：

```text
00_材料导读.md：学会整套材料怎样读、卡住时回哪里。
01_基础画像.md：学会老师身份、资料边界、履历和论文集合怎样读。
02_领域地图.md：学会导师方向处在哪片领域、旁边哪些方向容易混淆、先抓哪些概念。
03_论文路线.md：学会论文群怎样形成问题链，而不是背论文清单。
04_学习向导.md：学会从基础课走到目标论文、核心图、平台链路和进组前问题。
```

任何一轮如果开始围着单个例子打转，应立刻回到当前原文的整文学习地图。

## 每个目录放什么

| 目录 | 用途 | 平时怎么看 |
|:---|:---|:---|
| [00_入口/](00_入口/) | 当前学习路线和方向总览 | 每次回来继续学，先看这里 |
| [01_原文/](01_原文/) | 从正式成品复制来的 `00-04` 原文快照 | 需要核对事实或原文表述时再看 |
| [02_分讲/](02_分讲/) | 真正给人顺读的学习讲义 | 主阅读区，每次学习优先看这里 |
| [03_学习记录/](03_学习记录/) | 理解验收、问答记录、概念校准 | 追溯某个判断怎么来的时候再看 |
| [90_规范与历史/](90_规范与历史/) | 学习区使用规范、迁移对照、历史版本 | 只用于看懂这个学习区怎么使用 |

## 统一命名规则

`第 N 讲` 只用于能顺着读的学习材料。它要像一段讲解，解决一个问题，读完能多说清一句话。

`第 N 轮` 只用于学习记录。它可以保留你的回答、卡点、校准和验收，但不当作主阅读材料。

`01_原文/` 里的五份文件只作为学习时使用的正式材料快照。它们来自正式输出目录，但这里不负责替代正式成品，也不自动证明之后的输出已经同步。

这个文件夹内不记录项目设计事项。任何关于 `quality-workbench`、模板、verifier、harness、skill 的开发判断，都不放进 `张鹏举学习/`。

## 规范边界

旧路径迁移、历史入口和学习区写作规范都在 [90_规范与历史/](90_规范与历史/) 里。学习时不要先看这个目录。

以后不要再把原文、讲义、记录、旧入口和规范文件混在同一层。入口区只放当前会用的入口；过时但有追溯价值的学习材料，一律放进 `90_规范与历史/历史版本/`。这个文件夹的价值不在于材料多，而在于每次回来都知道自己该读哪一份。
