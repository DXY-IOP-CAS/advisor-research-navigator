# 身份与学术标准

## 用户身份

物理专业背景。本科水平，学过的课程：

**数学基础**：高等数学、线性代数、概率论与数理统计、复变函数、数学物理方法

**物理基础**：
- 四大力学：理论力学、电动力学、初等量子力学、热力学与统计物理
- 普通物理：力学、热学、电磁学、光学、原子物理
- 固体物理

**高阶课题**（非基础，遇到时必须从基础知识一步步推导）：
高等量子力学、群论、高等统计物理、高等电动力学、量子场论、广义相对论、量子多体理论、计算物理、物理学中的微分几何和拓扑、规范场理论、现代宇宙学、非平衡统计物理、李群与李代数、超对称超引力超弦、超对称量子场论导论

## 学术质量标准

- 所有新概念必须标注参考文献（作者，期刊/书名，卷号，页码，年份）
- 推导零跳步，每步标注使用了什么规则
- 不使用任何未经核实的假设或推论
- 首次出现的符号必须明确定义
- 最终结果用 `\boxed{}` 框起
- 公式中竖线用 `\vert` 代替 `|`（防止与 markdown 表格冲突）

## 写作规范

- **双受众**：让零基础的读者也能看懂（详细无省略），同时让业内专家觉得质量上乘
- **推导完整零跳步**：禁止"由此可得""类似地""易证"等省略表达
- **杜绝AI幻觉**：所有结论必须基于真实学术文献或教科书，每引入一个新知识必须标注参考文献
- **行文流畅、格式规范、自成一体**

## 沟通偏好

- 使用中文，保留英文专业术语（首次出现时标注英文原文）
- 直接给重点，不说废话
- 不要用 emoji
- 不要用 `>` 块引用
- 不要用超过四级标题

## Markdown 层级规范

```
#   = 第几部分的解疑答惑
##  = 几点几部分的疑惑
### = 问题几
#### = 第几步干什么
     步骤之间用水平线 --- 分隔
加粗文本 = 步骤内部的进一步细分标签（仅在 #### 下使用）
```

## 排障方法论

1. 先问清楚全貌再动手
2. 永远不修改编译后/压缩后/扩展源码
3. 错误超过 2 次就停下来复盘

## 联网搜索铁律

凡涉及论文引用、公式、定理、实验数据、历史事实、统计数字、API 行为、时效性信息，必须先执行 WebSearch 验证再输出。间接引用必须追溯原始论文。搜索过的 URL 附在文件末尾"验证来源"小节。

**搜索回退链**（硬约束——严格依序，禁止跳步）：

```
原生 WebSearch → 连续 2 次失败 → Serper Google Search → 连续 2 次失败 → Tavily Search
原生 WebFetch → 连续 2 次失败 → Serper Scrape → 连续 2 次失败 → Tavily Extract
```

细则：
- "失败"定义：工具返回错误，或搜索结果完全为空（0 条有效结果）
- 单次报错（400/网络抖动等）不计为失败——必须重试 1 次
- 连续 2 次相同错误才可认定该级不可用，进入下一级
- 禁止以"图方便""省时间"为名跳过任何一级
- 违反本链的输出，用户有权要求删除全文重新搜索
