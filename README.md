## 🛠️ 项目定位

`advisor-research-navigator` 是一个**面向准研究生设计的导师研究方向解构工具**。
由于从本科阶段（地大物理）跨越到前沿的超快物质科学与阿秒物理存在显著的知识断层，传统的粗泛阅读容易陷入“见林不见树”的困境。本项目通过结构化的数据抓取、论文关联图谱与学习桥接设计，帮助自己：
1. **锁准事实基础**：基于多源数据（官网、Google Scholar、OpenAlex）进行身份与履历的自动化核查。
2. **理清问题链路**：将导师多年发表的文献解构为由浅入深的问题推进线索，了解每一篇论文在课题组整体布局（HHG 平台搭建、方法学创立、液体水动力学观测）中的位置。
3. **精准倒推缺口**：对比自身知识背景与前沿文献所需技术栈，设计出极具实操性的、倒推至课本层级的进组自学路线图。

## 📈 个人学习进展

我正在按照本项目倒推的自学路线积极补足理论与技术背景：
- **第一阶段（已完成）**：搭建了导师信息与代表作的事实地图，初步消化了 2022 年 PRL 论文（首次证实液态水中的分子间库仑衰变 ICD 过程）。
- **第二阶段（进行中）**：深入研读 2025 年 Nature Communications 论文，理解 ICD 与质子转移、非绝热弛豫的竞争机制，并开始学习液相高次谐波产生（HHG）及液相磁瓶时间分辨光电子谱的仪器的物理原理。
- **下一步计划**：进入组内实习，在学长学姐和老师的指导下，将理论自学转化为实验与仿真工作。
