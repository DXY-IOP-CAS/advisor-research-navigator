# advisor-research-navigator

导师研究方向解构与自学路线梳理工具。

## 简介

本项目是一个自动化的工作流工具，旨在帮助准研究生在进入课题组前，系统性地梳理导师的研究脉络，并以此倒推个人的先修学习路线。

它将零散的公开资料进行结构化提取与分析，最终输出五份标准的 Markdown 文档，形成从“事实收集”到“学习指南”的完整闭环。

## 核心产出 (00-04 工作流)

通过该工具运行目标导师的公开信息，将自动生成以下五份核心报告：

1. **00_材料导读**：确定课题组的定位、核心术语及资料来源核查。
2. **01_基础画像**：建立导师的学术履历与代表性成果底座。
3. **02_领域地图**：梳理课题组所在研究领域的核心技术与应用背景。
4. **03_论文路线**：串联导师历年代表作，解构其研究问题的演进链条。
5. **04_学习向导**：将前沿论文中的实验手段与理论要求，倒推为基础课程与教材的自学清单。

## 目录结构

* `src/phase1/`：数据获取与处理核心脚本。
* `quality-workbench/`：文档标准与生成方法的测试工作台。
* `output/`：工具生成的导师调研产出目录。
* `张鹏举学习/`：个人自学笔记与概念校准区。

## 项目意义

阅读前沿文献时，初学者极易陷入“见林不见树”的困境。本工具的核心理念是：用系统化的逻辑强迫自己建立结构化的直觉。工具只是脚手架，真正的目的是利用它生成的路线图，扎实推进专业基础与前沿方法的自学进程。
