# domain-trace（阶段 2 — 领域脉络）

## 本 skill 是什么

输入：`01_基础画像.md`（含研究方向标签、代表性论文列表）
输出：`项目/导师/<姓名>/02_领域脉络.md`

任务：追溯该研究方向所属子领域的发展脉络——从历史起点到当前前沿。

## 工作流概览

```
输入：阶段 1 的画像 + 代表性论文
              │
              ▼
┌─────────────────────────────────────────────────┐
│ Step 1: 提取研究主题                              │
│  从画像提取研究方向关键词、代表性论文的标题/摘要  │
│  → 主题向量                                       │
└─────────────────────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────────┐
│ Step 2: 引文网络分析                              │
│  对代表性论文：                                     │
│   · 反向追溯：被哪些论文引用（找后续发展）         │
│   · 正向追溯：参考了哪些论文（找前驱）             │
│  跨源：OpenAlex + Semantic Scholar + INSPIRE      │
└─────────────────────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────────┐
│ Step 3: 里程碑论文筛选                             │
│  按影响力 + 时代分界筛选 5-15 篇里程碑            │
│  每篇写一句话贡献说明                              │
└─────────────────────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────────┐
│ Step 4: 领域树构建                                 │
│  把里程碑论文按时间/技术分岔组织成树形              │
│  大领域 → 子领域 → 当前分支                       │
└─────────────────────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────────┐
│ Step 5: 输出 02_领域脉络.md                        │
│  含领域树图、里程碑论文表、当前前沿定位            │
└─────────────────────────────────────────────────┘
```

## 输出文件结构

参见 `assets/02_领域脉络.md` 模板。

要点节：
- **领域定位**（一句话回答"这是个大领域下的什么细分方向"）
- **领域发展树**（从起源到当前，文字版树状图）
- **里程碑论文表**（标题/作者/年份/期刊/贡献一句话）
- **当前前沿定位**（导师的论文集合在哪一分支）
- **关键概念词典**（领域核心术语，每个一句话定义）

## 前置依赖

- `01_基础画像.md` 必须存在并已被用户审阅
- 代表性论文至少有 5 篇

## 调用脚本

| 脚本 | 用途 |
|:-----|:-----|
| `scripts/extract_themes.py` | 从画像+论文标题摘要提取主题关键词 |
| `scripts/citation_network.py` | 正向+反向引文分析 |
| `scripts/select_milestones.py` | 影响力+时间筛选 |
| `scripts/build_domain_tree.py` | 构建领域树 |
| `scripts/build_domain_trace.py` | 合成 02_领域脉络.md |
