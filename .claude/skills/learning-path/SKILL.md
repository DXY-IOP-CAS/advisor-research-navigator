# learning-path（阶段 4 — 学习讲义）

## 本 skill 是什么

输入：`01_基础画像.md` + `02_领域脉络.md` + `03_论文定位.md`
输出：`项目/导师/<姓名>/04_学习讲义.md`

任务：从**高等数学 + 线性代数**起步，反推一条知识链，让用户走通就能跟上导师的前沿研究。

## 工作流概览

```
输入：前三阶段产出
              │
              ▼
┌─────────────────────────────────────────────────┐
│ Step 1: 列举领域核心知识点                         │
│  从 02 领域脉络的"关键概念词典"开始                 │
│  从 03 论文定位的"前沿位置"提取新知识点              │
│  从 01 画像的"研究方向"找出领域技术词汇             │
└─────────────────────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────────┐
│ Step 2: 反推依赖链                                 │
│  每个目标知识点 → 它的先修知识 → 先修的先修 → ...   │
│  直到基础是高数/线代/普物                           │
└─────────────────────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────────┐
│ Step 3: 章节路径排序                               │
│  按依赖链的拓扑序组织章节                           │
│  每章节:知识点 + 推荐教材 + 推荐课程 + 习题建议    │
└─────────────────────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────────┐
│ Step 4: 路径索引                                   │
│  教材链接、课程视频（B站/Coursera）、综述论文      │
└─────────────────────────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────────┐
│ Step 5: 输出 04_学习讲义.md                        │
│  遵循学习讲义写作规范（叙事流+排版）                │
└─────────────────────────────────────────────────┘
```

## 输出文件结构

参见 `assets/04_学习讲义.md` 模板。

要点节：
- **起点声明**（从高数+线代起步）
- **领域全景图**（一张图说明学到尽头能做什么）
- **章节递进路线**（每章节含先修、本章、目标、教材）
- **关键概念词典**（按学习顺序排列）
- **资源索引**（教材链接、视频、综述）
- **预计学习时间**（每章节大小时长）

## 写作规范

详见 `references/pedagogy-rules.md`

核心约束：
- **零跳步**——每章节只引用前一章节讲过的概念
- **不写教材正文**——给路径给索引，不替代教科书
- **大字典做参照系**——以经典教材为坐标
- **资源必须可验证**——所有视频/课程链接访问过再写入

## 决不做什么

- ❌ 不写"这是 XXX 定理的证明"——给定义给直觉，不给推导
- ❌ 不写习题答案
- ❌ 不给具体研究建议——"是否读博"用户自己决定
- ❌ 不评论老师——价值判断用户做

## 前置依赖

- `01_基础画像.md`、`02_领域脉络.md`、`03_论文定位.md` 都必须存在
- 写作规范文档 `references/pedagogy-rules.md` 必须存在（即使只是骨架）

## 调用脚本

| 脚本 | 用途 |
|:-----|:-----|
| `scripts/extract_concepts.py` | 从前三阶段抽取核心概念 |
| `scripts/build_dependency_graph.py` | 反推依赖链 |
| `scripts/chapter_sorter.py` | 拓扑排序章节 |
| `scripts/resource_lookup.py` | 查教材+视频+综述链接 |
| `scripts/build_learning_path.py` | 合成 04_学习讲义.md |
