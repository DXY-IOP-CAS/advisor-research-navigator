# 付费方案备忘 — 规模化扩展场景

**生成日期**：2026-07-01
**背景**：当前物理所~80 位导师规模不需要付费，$0 方案（curl_cffi + Playwright 兜底）足够。如未来要扩展成面向全中国高校/科研机构的网站，以下付费渠道可作为基础设施层备选。

---

## 1. 付费渠道全景

### 1.1 SERP API（GS 数据代理）

最直接：服务商维护代理池 + 浏览器渲染 + CAPTCHA 破解，你只需发 HTTP 请求。

| 服务 | 起步价 | 免费额度 | 适合规模 |
|:-----|:------|:---------|:---------|
| **SerpAPI** | $75/月 (5K 查询) | 100 次/月试用 | 起步商业 |
| **ScaleSERP** | $66/月 (未公开) | 无免费层 | 中型 |
| **ScraperAPI** | $49/月 (5K 页) | 1K 页/月免费 | 单个项目 |
| **SearchAPI** | $50/月 (1K 次) | 100 次/月 | 小规模 |
| **Scrapeless** | $59/月 | 无免费层 | 中型 |
| **BrightData** | $500+ /月 | — | 企业级 |

**点评**：SerpAPI 最成熟（GS 专用 Scholar engine，返回结构化 JSON），文档完善，价格中档。如果规模到 50 万次/月需要谈企业折扣（1-2 美分/次）。

SerpAPI：https://serpapi.com/google-scholar-api
ScaleSERP：https://scaleserp.com/google-scholar-api
Scrapeless：https://www.scrapeless.com/

### 1.2 付费住宅代理

Playwright + 代理池方案适合对 GS 大规模并发请求（不是单 IP 慢跑）。

| 服务 | 起步价 | 流量 | 特点 |
|:-----|:------|:-----|:-----|
| **BrightData** | $500/月 | 5GB 住宅代理 | 最大代理网络，IP 池 7200 万 |
| **Oxylabs** | $300/月 | 4GB 住宅代理 | 第二大方，欧洲覆盖好 |
| **Smartproxy** | $75/月 | 3GB 住宅代理 | 性价比高，适合中小规模 |
| **Webshare** | $70/月 | 7500 代理 | 固定代理，非轮换 |
| **Soax** | $99/月 | 3GB 住宅代理 | 位置精确到城市 |

**点评**：如果 curl_cffi + Playwright 对 GS 生效，代理不是必需品——限速比 IP 轮换更关键。只有需要同时对几十个 GS profile 高并发时才需要代理。

但注意：GS 对住宅 IP 的限制比数据中心 IP 宽松得多。当前我们的 Windows 11 家庭网络本身就是住宅 IP，这是优势而非问题。

BrightData：https://brightdata.com/
Oxylabs：https://oxylabs.io/
Smartproxy：https://smartproxy.com/
Webshare：https://www.webshare.io/
Soax：https://soax.com/

### 1.3 付费学术数据库 API

GS 之外的补充数据源（部分阶段 2/3 可能用到），走官方渠道：

| 数据库 | 费用 | 用途 |
|:-------|:-----|:------|
| **Scopus API** | 免费但需机构订阅（Elsevier 年费） | 引用分析、学科筛选 |
| **Web of Science API Starter** | 免费用量极低，扩展需 Clarivate 报价 | 文献索引黄金标准 |
| **IEEE Xplore API** | 免费（注册后 200 次/天） | CS/工程论文 |
| **INSPIRE-HEP** | 免费 | 高能物理事实标准 |
| **NASA ADS** | 免费（需注册 API token） | 天体物理 |
| **CNKI 知网** | 需机构订阅或按篇付费 | 中文学位论文/期刊 |

**点评**：阶段 1 的 GS + OpenAlex + arXiv 三源组合已经覆盖 90%+。需要付 Scopus/WoS 的场景是阶段 2（领域脉络分析）和阶段 3（论文串联定位）——这些 API 提供更规范的主题分类和引用网络。

### 1.4 全托管学术搜索 API

这些不是 GS 替代，而是让 AI Agent 不用自己写爬虫的多源搜索层。

| 服务 | 费用 | 覆盖源 |
|:-----|:------|:-------|
| **学术文献检索 MCP**（ai.boce.com） | 免费开源 | 13+ 源（arXiv, OA, S2, PubMed, Crossref 等） |
| **SouWen（搜文）** | 免费开源 | 91 源（含学术 17 + 专利 8 + 网页 29） |
| **paper-search MCP** | 免费开源 | 20+ 学术源 |
| **opencite** | 免费开源 | 10+ 源 + CLI + BibTeX |
| **academic-mcp** | 免费开源 | DeepXiv + CNKI + WoS |
| **RefTracker** | 免费（有付费版） | 引用追踪 |
| **ScholarAPI**（scholarapi.net） | 付费，价格未公开 | 百万篇 OA 全文 + 元数据 |

这些工具大多是 Python 库或 MCP Server，免费跑在你自己的机器上。它们不解决 GS 封锁（因为 GS 不是它们的源），但可以丰富阶段 2/3 的引用网络数据。

## 2. 分规模费用估算

| 规模 | 导师数 | 月请求量 | 适合方案 | 月费 |
|:-----|:------|:---------|:--------|:----:|
| **单项目** | ~80 人 | < 5000 次/月 | $0 方案（curl_cffi + Playwright 兜底 + 限速） | $0 |
| **小型机构** | ~500 人 | ~3 万次/月 | $0 方案 + 代理池（Smartproxy $75） | $75 |
| **中型机构** | ~2000 人 | ~12 万次/月 | SerpAPI + 代理池（$75 + $150） | $225 |
| **全国平台** | ~5000+ 人 | ~30 万次/月 | SerpAPI 企业套餐或自建爬虫云（~$500-1000） | $500+ |

### 费用分解（全国平台 5000 人场景）

```
每人每次 GS 刷新 ~ 30 次请求（~5 页翻页 × 6 数据源）
月刷新一次 → 5000 × 30 = 150,000 次请求

方案 1：SerpAPI 企业套餐
  $0.02/次 × 150,000 = $3,000/月（谈判后可降）

方案 2：自建 curl_cffi + Playwright 集群
  5 台低配云服务器 × $30/月 = $150/月
  + BrightData 代理 $500/月（可选，非必须）
  总计 $150-650/月

方案 3：ScraperAPI 通用代理
  $49/月 × 20 个账号 = $980/月
  质量不如 SerpAPI 专用 endpoint
```

## 3. 付费 vs 免费的核心决策点

| 维度 | 免费方案（curl_cffi + Playwright） | 付费方案（SerpAPI） |
|:-----|:------------------------------------|:---------------------|
| 初始成本 | $0 | $75/月起步 |
| 维护成本 | 随 GS 反爬更新迭代 | 0 维护（服务商负责） |
| 请求限制 | 2 req/min 单 IP | ~30 req/min |
| 数据质量 | 原始 HTML，需自己解析 | 结构化 JSON |
| 可用性 | 偶尔被封（1-24h 恢复） | 99.9%+ |
| 法律风险 | 违反 GS ToS | 服务商承担风险 |
| 扩展瓶颈 | 单机单 IP | 按量付费无上限 |

## 4. 推荐扩展路径

```
阶段 1（物理所 80 人，当前）：
  $0 方案 → 验证工具价值和输出质量
  不看付费方案，不花一分钱

阶段 2（扩展到 3-5 个物理所/清华/南大等，~500 人）：
  如果 $0 方案稳定运行 → 继续保持
  如果被封频发 → SerpAPI 入门套餐 $75/月兜底部分导师

阶段 3（全国平台，5000+ 人）：
  平台有收入后再考虑付费
  推荐 SerpAPI 企业套餐（量大可压价到 $0.01/次）
  或 BrightData 代理 + 自建爬虫集群（月费 $500-1000 vs 代收分成收入）
  不要一开始就铺基础设施
```

## 5. 付费策略建议

1. **先零后付**。在物理所 80 人规模验证全流程，$0 方案能跑就不付钱。
2. **付值不付功能**。不要为"万一以后需要"功能付费。有 200 万用户用 PoP 的 $0 路径。
3. **SerpAPI 是唯一值得付的 GS 入口**。其他 SERP API 要么质量不如它，要么比它贵。唯一的例外是你技术实力强到自建 curl_cffi 代理池。
4. **大规模时自建比 API 便宜**。150,000 次/月用 SerpAPI 要 ~$3,000，自建集群 + 住宅代理约 $150-650，差 5-20 倍。但自建需要运维能力。
5. **Scopus/WoS 在阶段 2/3 再考虑**。阶段 1 不需要它们。阶段 2（领域脉络分析）可能需要 WoS 的分类标签，但 OpenAlex 的学科字段够做基础分析。

所有 URL 和价格为 2026-07 调研结果。价格可能随时间变化。
